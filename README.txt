This ZIP contains 4 different sections detailed below in file order:

------------------------------------------------------------------------------

LaTeX Code: Folder containing LaTeX code, figures, and bibliography 
to generate the project report output file.

------------------------------------------------------------------------------

src: Folder containing 2 subfolders: functions and main.

functions: Folder containing 4 self-made R functions
used in the data analysis for the report.

main: Folder containing 2 R scripts: 

DataCleaning: responsible for transforming Marine_CSV_sample.csv into cleanedMarineData.csv

DataAnalysis: responsible for the figures, modeling, and data analysis
presented in the report.

------------------------------------------------------------------------------

volume: Folder containing 2 subfolders: data and figures.

data: Contains uncleaned data (Marine_CSV_sample.csv)
and cleaned data after data cleaning (cleanedMarineData.csv)

figures: Contains 9 generated figures used in the report.

------------------------------------------------------------------------------

Maritime-Weather-Forecast-Report.pdf: The Report (pdf).
